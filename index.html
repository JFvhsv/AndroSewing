Обзор приложения

AndroSew (FL-GUI-Tools) - это графический интерфейс для работы с Android-устройствами с использованием инструментов ADB (Android Debug Bridge) и Fastboot. Приложение позволяет выполнять различные операции с устройствами, такие как отправка файлов, перезагрузка в различные режимы, выполнение команд и многое другое.

Основные возможности приложения:

  • Управление устройствами через ADB
  • Управление устройствами через Fastboot
  • Выполнение команд ADB и Fastboot
  • Перезагрузка устройств в различные режимы
  • Очистка разделов устройства
  • Отправка файлов на устройство
  • Получение информации об устройстве

Автопрошивка

Окно автоматической прошивки позволяет выполнять прошивку устройств по заранее заданным сценариям. Это упрощает процесс и минимизирует риск ошибок.

Основные функции

Выбор сценария

Позволяет выбрать заранее подготовленный сценарий прошивки.

Запуск прошивки

Запускает процесс прошивки устройства в соответствии с выбранным сценарием.

Мониторинг процесса

Отображает ход выполнения прошивки и выводит сообщения об ошибках, если они возникают.

Примечание: Перед началом прошивки убедитесь, что устройство подключено и находится в нужном режиме: загруженно в операционную систему!

Главное окно

Главное окно приложения предоставляет доступ ко всем основным функциям через панель инструментов и контейнер для отображения других окон.

Панель инструментов

Панель инструментов содержит кнопки для быстрого доступа к различным функциям:

Командная строка

Открывает командную строку Windows для выполнения команд вручную.

ADB инструменты

Открывает окно с инструментами ADB для работы с подключенными устройствами.

Fastboot инструменты

Открывает окно с инструментами Fastboot для работы с устройствами в режиме Fastboot.

Документация

Открывает встроенную документацию для помощи в использовании приложения.

Настройки

Открывает окно настроек приложения для изменения параметров работы.

Примечание: В нижней части главного окна расположен индикатор прогресса, который отображается при выполнении длительных операций.

ADB инструменты

Окно ADB инструментов предоставляет доступ к функциям Android Debug Bridge для взаимодействия с подключенными устройствами.

Список устройств

В верхней части окна отображается список подключенных устройств с возможностью обновления.

Основные функции

Отправка файлов

Выберите файл на компьютере и отправьте его на устройство.

Использование:

  1. Нажмите кнопку "Обзор" для выбора файла
  2. Выбранный путь отобразится в текстовом поле
  3. Нажмите кнопку "Отправить"
Перезагрузка устройства

Перезагружает устройство в нужный режим.

Использование: Нажмите кнопку "Перезагрузить в `РАЗДЕЛ`"

Режим Sideload

Запускает экран Sideload для установки OTA-обновлений. Для загрузки прошивки используйте команду adb sideload <путь_к_файлу>.zip.

Использование: Нажмите кнопку "Sideload Screen"

Очистка разделов

Позволяет очистить различные разделы устройства.

Использование:

  1. Выберите раздел из выпадающего списка
  2. Нажмите кнопку "Очистить"

Выполнение ADB команд

В нижней части окна находится раздел для ввода и выполнения произвольных ADB команд.

Использование:

  1. Введите команду в текстовое поле
  2. Нажмите кнопку "Отправить"
  3. Результат выполнения команды отобразится в списке ниже

Важно: Для работы ADB инструментов необходимо, чтобы на устройстве была включена отладка по USB в настройках разработчика, а на компьютере установлены драйверы устройства.

Команда Описание
adb devices Список подключенных устройств
adb push <источник> <назначение> Отправить файл на устройство
adb pull <источник> <назначение> Скачать файл с устройства
adb reboot Перезагрузить устройство
adb reboot recovery Перезагрузить в режим восстановления
adb shell Запустить командную оболочку на устройстве

Fastboot инструменты

Окно Fastboot инструментов предоставляет функции для работы с устройствами в режиме Fastboot, который используется для прошивки и восстановления устройств.

Список устройств

В верхней части окна отображается список подключенных устройств в режиме Fastboot с возможностью обновления.

Получение переменных

Позволяет получить значения различных системных переменных устройства.

Получение переменной

Получение значения конкретной переменной.

Использование:

  1. Введите имя переменной в текстовое поле
  2. Нажмите кнопку "Запросить"
  3. Результат отобразится в списке ниже
Быстрый запрос слотов

Получение значений часто используемых переменных: устройство имеет 2 раздела (A/B) или 1 (A) .

Использование: Нажмите кнопку "Быстрый запрос слотов"

Запуск образов в ОЗУ

Позволяет загрузить и запустить образы прошивки в оперативной памяти устройства.

Загрузка образа загрузчика (Boot)

Загрузка образа загрузчика в ОЗУ устройства.

Использование:

  1. Нажмите кнопку "Обзор" для выбора файла образа
  2. Нажмите кнопку "Запустить"
Загрузка других образов

Загрузка других образов системы (recovery, dtbo и т.д.).

Использование: Аналогично загрузке образа загрузчика

Прошивка разделов

Прошивает выбранный раздел

Использование: Аналогично запуску образа в ОЗУ

Перезагрузка устройства

Перезагрузка устройства

Перезагружает устройство в нужный режим.

Использование: Нажмите кнопку "Перезагрузить в `РАЗДЕЛ`"

Выполнение Fastboot команд

В нижней части окна находится раздел для ввода и выполнения произвольных Fastboot команд.

Использование:

  1. Введите команду в текстовое поле
  2. Нажмите кнопку "Отправить"
  3. Результат выполнения команды отобразится в списке ниже

Важно: Для работы с Fastboot инструментами устройство должно быть загружено в режим Fastboot (обычно достигается одновременным нажатием кнопок питания и громкости вниз при выключенном устройстве).

Команда Описание
fastboot devices Список подключенных устройств в режиме Fastboot
fastboot getvar <переменная> Получить значение переменной
fastboot flash <раздел> <файл> Прошить раздел устройства указанным файлом
fastboot boot <образ> Загрузить образ в ОЗУ и запустить его
fastboot reboot Перезагрузить устройство в обычный режим
fastboot reboot recovery Перезагрузить устройство в режим восстановления
fastboot reboot bootloader Перезагрузить устройство в режим загрузчика

Документация

Окно документации предоставляет доступ к встроенной документации по использованию приложения через встроенный веб-браузер.

Примечание: Для отображения документации используется встроенный браузер Chromium (CefSharp), который загружает документацию из указанного в настройках URL.

Настройки

Окно настроек позволяет настроить различные параметры работы приложения.

Доступные настройки:

Анимация

Включение или отключение анимации в интерфейсе.

Путь к инструментам

Указание пути к директории с инструментами ADB и Fastboot.

Запуск утилит от имени администратора

Включение запуска утилит с правами администратора.

Ссылка на документацию

URL для загрузки встроенной документации.
